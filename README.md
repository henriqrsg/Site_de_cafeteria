# Cafeteria Website

Projeto de um site simples para uma cafeteria, desenvolvido utilizando **HTML5** e **CSS3**, com foco em estrutura semântica, organização visual e responsividade básica.

## Objetivo

Este projeto foi criado com o intuito de praticar:

- Estruturação de páginas com HTML5 semântico  
- Estilização moderna com CSS3  
- Organização de layout e hierarquia visual  
- Boas práticas de front-end básico  

## Tecnologias Utilizadas

- HTML5  
- CSS3  

## Funcionalidades

- Página inicial com apresentação da cafeteria  
- Seção de produtos (cafés, bebidas, etc.)  
- Layout limpo e intuitivo  
- Navegação simples entre seções  
- Estilização personalizada  

## Destaques Técnicos

- Uso de tags semânticas (`<header>`, `<section>`, `<footer>`)
- Separação clara entre estrutura (HTML) e estilo (CSS)
- Organização modular de arquivos
- Layout pensado para boa legibilidade
